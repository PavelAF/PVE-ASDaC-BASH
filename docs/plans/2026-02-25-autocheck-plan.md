### 2. Функция `exec_agent_cmd` — выполнение через Guest Agent

**Файл:** `PVE-ASDaC-BASH.sh`

Аргументы: `vmid`, `node`, `command`
Возвращает: вывод команды через nameref

Алгоритм:
1. `POST /nodes/{node}/qemu/{vmid}/agent/exec` с `command=<cmd>`
2. Из ответа извлечь `pid`
3. Цикл поллинга (до 10 сек, интервал 1 сек):
   `GET /nodes/{node}/qemu/{vmid}/agent/exec-status?pid={pid}`
4. Из ответа извлечь `out-data` (stdout) и `err-data` (stderr)
5. Вернуть объединённый вывод или код ошибки

### 3. Функция `exec_serial_cmd` — выполнение через serial console

**Файл:** `PVE-ASDaC-BASH.sh`

Аргументы: `vmid`, `command`, `prompt_regex`, `timeout`, `login`, `password`
Возвращает: вывод команды через nameref

Алгоритм:
1. Проверить наличие сокета `/var/run/qemu-server/{vmid}.serial0`
2. Через socat подключиться к сокету
3. Отправить Enter, прочитать ответ
4. Если login-промпт — отправить логин, пароль
5. Отправить команду
6. Ждать `timeout` секунд, захватить вывод
7. Очистить вывод (убрать эхо команды, промпт)
8. Вернуть результат

### 4. Функция `autocheck_stand` — основная логика

**Файл:** `PVE-ASDaC-BASH.sh`

Аргументы: `config_file` (опционально, если не задан — интерактивный выбор)

Алгоритм:
1. Если `config_file` не задан:
   - Сканировать папку `autocheck/` рядом со скриптом
   - Для каждого `.conf` — `source` и вывести `autocheck_name`
   - Пользователь выбирает конфиг
2. `source` выбранный конфиг
3. Распарсить `autocheck_vms` — получить список ВМ и типов exec
4. Обнаружить стенды (переиспользовать логику `manage_stands`)
5. Пользователь выбирает стенд(ы)
6. Для каждого стенда:
   a. Получить список ВМ из пула (API `/pools/{pool}`)
   b. Построить маппинг: имя ВМ → vmid, node, status
   c. Вывести заголовок стенда
   d. Для каждого `check_N`:
      - Вывести заголовок проверки (`check_N_name`)
      - Для каждой ВМ из `check_N_vms`:
        * Найти vmid по имени в маппинге
        * Если ВМ не найдена/не запущена — предупреждение, пропуск
        * Определить exec_type из `autocheck_vms`
        * Выбрать команду: `check_N_cmd_{exec_type}`
        * Вызвать `exec_agent_cmd` или `exec_serial_cmd`
        * Вывести результат
